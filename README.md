# eddb-sync
 Sync Data from EDDB for Ship Shopper

 Requires NodeJS
 - Install NodeJS
 - Navigate to project directory in Terminal
 - Run `npm install`
 - See scripts below

 ## Scripts
 `npm run fetch`
 Fetch raw data from EDDB and cache to local files (/raw-data)

 `npm run process`
Process cached data to minify (/processed-data)

`npm run full`
Both of the above
